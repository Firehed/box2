Box

Box is a library and command line application for simplifying the PHAR creation process.

Download .zip Download .tar.gz View on GitHub

Installation

To install, you need to run this command in your shell:

$ curl -s http://kherge.github.com/Box/installer.php | php

Updates can later be performed by running Box's update command:

$ php box.phar update

Usage

To build a new PHAR using Box, configure your project

box.json:

{
    "directories": [
        "/path/to/source",
        "/path/to/source"
    ]
}

and run

$ php box.phar build

You can also check out the example PHAR application ready to be build by Box.

Please see the wiki for more detailed usage information.
